# Community Scope

## Included

### Notes

- rich note editing
- notebooks
- file attachments needed by notes
- note version restore only if required by the editor split

### Team

- signup/login/setup
- organization bootstrap
- member list
- invite and role management

### Search

- embedding generation and reindex
- full-text plus embedding search if needed for UX continuity
- search index status

### Graph

- global note graph
- graph search
- graph settings required for visualization

### Backup

- snapshot creation
- destination management
- encryption passphrase
- restore flow

## Excluded

- conversational AI
- writing/review agents
- paper/patent workflows
- OCR and vision pipelines unless a retained feature hard-depends on them
- compliance suite
- appliance factory workflow
- in-app feedback
- todo and protocol systems
- external scholarly integrations

## Rule

If a module exists only to support an excluded feature, do not import it into this repo.
